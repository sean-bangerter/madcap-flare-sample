STEP 5: Add a Logo and a Map to the Self-service Portal

Logo

  1. Log in to the Printer Installer server.
  2. Open Windows Explorer.
  3. Navigate to c:\inetpub\wwwroot\public\portal.
  1. Add your new image file into the portal folder.

    Option 1: Overwrite the existing default logo file

    1. Save your image file as company-logo.png to c:\inetpub\wwwroot\public\portal.
    2. Open the Self-service Portal, which will display the updated image file.

    Option 2: Use an image with a different file name in the same location as the default file

    1. Save an image file with any name (other than company-logo.png) to c:\inetpub\wwwroot\public\portal.
    1. In c:\inetpub\wwwroot\public\portal, open header.html.
    2. Change the default file name (company-logo.png) to the new file name.
    1. Save header.html.
    2. Open the Self-service Portal to see if your new image appears as needed.

Map

  1. In the Admin Console, click Return to tree view if needed.
  1. In the tree view, select the folder that corresponds to a specific location.

For example, let's say you have an office in Chicago. As such, you create a folder for Chicago in the tree view, which is selected below.

  1. Click the Map tab.
  2. When the Map tab appears, click Upload Background Image.
  1. Click Choose File.
  2. Select the image you want to upload and click Open.
  1. When the background image has been chosen, click Upload Image.

The image will appear in Printer Installer as shown below.

  1. From the gray box, drag-and-drop a printer onto your map.
  1. After you have added the printers to the map, click Save.
