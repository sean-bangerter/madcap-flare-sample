OM Plus Printing

OM Plus is primarily a back-end print management solution from Plus Technologies. OM Plus basically takes print jobs and makes sure they print where, when, and how the customer needs them to. Because OM Plus does not support the management of actual drivers and print queues, their platform integrates well with Printer Installer to provide a comprehensive printing solution.

For the OM Plus options to appear in the Admin Console, you must have a license that supports OM Plus. When you have the license set up, all you need to do is replace your existing license with your new one. For more information, see STEP 3: Apply your License to your Installation.

With the proper license added to your Printer Installer setup, you will see several features added to the Admin Console interface. First, if you select a folder in the tree view and then click New, the option for OM Plus Managed Printer will appear. The purpose of this option is to enable you to add a new OM Plus Managed Printer object to the selected folder in the tree view. Setting up this type of printer object uses a similar setup window used for any TCP/IP printer (via the Add OM Plus Manged Printer window), and the new OM Plus Manged Printer object appears in the tree view (this type of object uses a different icon than the TCP/IP printer). Moreover, if you set up the new OM Plus Managed Printer through the tree view, then the Port tab will show the name you have given the printer in the Queue Name field.

In the example above, the OM Plus Managed Printer was added from the tree view and named TestPrinter. As such, the Port tab shows that same name in the Queue Name field. The name you use must meet all the requirements of OM Plus. You will notice that this name cannot be changed in the Port tab. Rather, click the General tab and change the name there.

If you would rather use an existing TCP/IP printer object in the tree view instead of creating a new OM Plus Managed Printer object, just select the printer object in the tree view, click the Port tab, enable OM Plus Delivery Manager (click Route all print jobs to OM Plus Delivery Manager), and enter the Queue Name.

Like many other features in the Admin Console, the OM Plus settings enable you to apply default settings or printer-specific settings to the OM Plus Managed Printer object. If you opt to use the printer-specific settings, select the OM Plus Managed Printer object in the tree view and then click the Port tab. In the OM Plus section, select Use printer-specific servers, click Add, and then enter the OM Plus server hostname, for example, omplus.demo.com.

If you opt to use the default settings, select the OM Plus Managed Printer object in the tree view and then click the Port tab. In the OM Plus section, select Use default servers and save your changes. Then, click Tools > Settings > General to configure the default settings. If you have not yet set any OM Plus servers, you will see a notification to information you that the servers are not configured. In this case, click Add and then enter the hostname of the OM Plus server.

Regardless of whether you are using the default settings or the printer-specific settings, you will need to enter the username and password credentials to communicate with the OM Plus server. Though these credentials appear in Tools > Settings > General under the OM Plus section, they apply to both default and printer-specific settings.
