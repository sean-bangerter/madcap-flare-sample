Google Cloud Printing Cheat Sheet

The Google Cloud Printing tab appears when you select a service host object in the tree view as shown in the image to the left.

When setting up Google Cloud Printing, just click Enable Google Cloud Printing and then click None Selected to select a printer object as the virtual pull print queue to process mobile print jobs. After you select a printer object, the None Selected text will change to show the name of the selected printer object.

Understanding the difference between Traditional and Local is important as well. If you do not want your jobs going through Google's severs, then select Local. This option is used for environments in which security is both elevated and crucial. If sending your print jobs to Google's servers is not an issue, click Traditional. Read More...

First, provide the Google Cloud Printer with a name and then apply your credentials to Google Cloud in the Google Authentication fields. Before you proceed, make sure you are logged in to the Google Cloud account with the credentials you have provided. Finally, click Register Printer to finish the setup. For more information, see Mobile Printing.

Enable Google Cloud Printing

Clicking this option enables Google Cloud Printing in your Printer Installer environment. If Google Cloud Printing is enabled, then Printer Installer will use the Pull Print Queue selected in the Google Cloud Printing tab. For more information, see Mobile Printing.

None Selected

Click None Selected to select the pull printer that will be used as the queue for mobile print jobs. Clicking this option opens the Select Mobile Pull Printer window (shown below), which enables you to select the pull printer that will be used as the queue.

Keep in mind that only the printer objects that have the Make Pull Printer field selected in the Printing tab will appear in the Select Mobile Pull Printer window. Once you select a printer object to be the queue, then the None Selected button will display the name of the printer object.

If a printer object has the Make Pull Printer field selected, and that printer object is selected for mobile printing (through the Mobile Pull Printer window), then the printer object will be designated for releasing mobile print jobs. Furthermore, if a printer object is set for pull printing and for mobile printing, then you cannot delete it from the tree view. You will receive a message stating that you cannot delete the node.

You must select a different printer object for mobile printing before you can delete the printer object.

Print Job Processing Type

Traditional

Select Traditional to create the mobile printer in the cloud and then share that printer with the users you would like to have it.

Local

Select Local to create the mobile printer that only the users who are on the same subnet as the service host can see; however, all jobs stay local and don't ever need to go to the cloud. If you select Local, the Register Printer button will be disabled because registration to the Google Cloud Print Admin system (www.google.com/cloudprint) is not required.

Google Authentication

Admin Email

In this field, enter the administrator-level email address for the Google Cloud Print account to which the documents will be published for printing.

Google Password

In this field, enter the password for the Google Cloud Print account to which the documents will be published for printing.

Use Proxy Server

Use these options if you have an environment that does not allow direct access to the Internet, and your computers must communicate via a proxy server, which is a "go-between" that may allow access to sites that otherwise could not be accessed.

Proxy Server

The web address used to access a proxy server through a web browser.

Proxy Server Port

The web address used to access a proxy server through a web browser.

Proxy Username (Optional)

The web address used to access a proxy server through a web browser.

Proxy Password (Optional)

The web address used to access a proxy server through a web browser.

Google Cloud Printer Registration

This button registers the selected pull printer with Google Cloud Print. Keep in mind that before you click Register Printer, you must log in to Google Cloud Print using the same credentials you entered in the Admin Email and Google Password fields.

NOTE: The Register Printer button will be disabled, by default, until you have properly filled in the required fields of the Google Cloud Printing tab.
