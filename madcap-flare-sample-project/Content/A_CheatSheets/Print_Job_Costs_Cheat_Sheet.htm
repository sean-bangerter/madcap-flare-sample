Print Job Costs Cheat Sheet

The Print Job Costs settings appear in the Admin Console when you click Tools > Print Job Costs. The image to the left shows the Print Job Costs settings as they appear in Printer Installer.

The Print Job Costs settings enable an administrator to set up costing templates, which Printer Installer can use to assign print costs by paper size and printing type. For example, five cents for printing black and white, single-sided. By default, the Print Job Costs provide one template (named Default Costs). Read More...

You can add as many templates as you need by clicking Add New Template. When you add a new template, you will be directed to give the template a name and a description. The new template will, by default, provide a line item for the Other paper type, which applies to any paper type not configured in the template. However, you can click the plus sign (+) icon to add new line items for each paper type to be configured. Any paper types not configured in the template will be reported as Other. Note that each field in a template must have a value, even if that value is 0.000. Also, make sure you save any line items individually in the template. When you are done setting up your templates, you can apply a template to a printer object via the General tab under the Print Job Costing field. The fields of the Print Job Costs settings are described in detail below.

Add New Template

Click Add New Template, and Printer Installer will open a drop-down to enable you to give a name and description to the template. When you are finished, click Save, and you will see the new template appear with the name and description you have provided.

Delete

Select a template name and click Delete to remove entire templates from the Admin Console. Printer Installer does not allow you to delete the out-of-the-box Default Costs template. Additionally, if you delete a template that is currently in use by a printer object then that printer object will automatically start using the Default Costs template.

Template Name

The Template Name column shows the name of each available template. When you create a template, the Template Name field will remain blank until you enter a template name and click Save.

Template Description

The Template Description column shows a description or purpose of each available template. When you create a template, the Template Description field will remain blank until you enter a description and click Save.

Default Costs

Default Costs is the out-of-the-box template that is available as a basic placeholder, but you can use it if you would like. Because it is the default template, it cannot be deleted from the Admin Console. There is a check box next to it, but that check box is disabled. When you have the Default Costs template selected, the details of that template only appear in the Print Job Costs section because details for only one template can appear at a time.

Paper Type

The Paper Type column shows the paper types that have been added to the template. Each paper type has fields for black and white and color for both single-sided and double-sided printing. All of these fields must have an entry, even if it is 0.00, for you to be able to save the paper type to the template. Paper Types are added to a template via the plus sign (+) button. Each template has this button to add paper types specifically to that template. Similarly, you can use the Remove button to eliminate a paper type from a template.

When you add values to a template, you then apply the template to one or more printer objects in the General Tab under the Print Job Costing section as shown below.

At this point, Printer Installer will be able to use the paper type values to track costs for each printer object using the template.
