Windows Print Server Links Cheat Sheet

The Windows Print Server Links appear in the Admin Console when you click Tools > Windows Print Server Links. The image to the left shows the Windows Print Server Links settings as they appear in Printer Installer.

The Windows Print Server Links settings are used for administrators who continue to use a print server, but they would still like to take advantage of Printer Installer's self-install or printer deployment capabilities. Click Add to set up a new Windows Print Server Link, enter the print server name (you can also navigate to the print server), and then select the tree view folder that will hold the printers on the print server. Read More...

A Windows Print Server Link will appear with the print server name, the total number of printers on the print server, the shared printers, and the printers that appear in the tree view as printer objects. A Windows Print Server Link also displays the tree view folder that stores the printers from the print server. The fields of the Windows Print Server Links settings are described in detail below.

Add

Click Add and the Add Windows Print Server window appears as shown below.

This window enables you to do two things - first, select the Windows Print Server you are using to manage your printers, and second, select a folder from the tree view to display the printers that reside on the Windows Print Server. When you have finished adding the Windows Print Server Link, then the tree view folder you selected will show the printers stored in the Windows Print Server. For more information, see Windows Print Server Links (WPSL).

Edit

Select a Windows Print Server Link from the list of the ones available and then click Edit. The Edit Windows Print Server window appears. You cannot edit the Windows Print Server selected, but you can edit the tree view folder in which the printers on the print server appears.

Delete

Select a Windows Print Server Link and click Delete to remove the item from the list of available links.

Print Server Name

The Print Server Name column displays the print server that you selected when you clicked Add as shown below.

The print server you select will appear in the Print Server Name column. Moreover, the printers on the server will appear as printer objects in the tree view folder you select in the List Printers In field.

Last Synchronization

The synchronization refers to instances in which the client that is installed on the Windows Print Server monitors the server for changes and reports them to Printer Installer.

Total Printers

The Total Printers column displays that total number of printers the client finds (shared or not) on the Windows Print Server. You may see a difference between Total Printers and Listed Printers because the Listed Printers does not show printers that are not shared.

Shared Printers

A printer residing on the Windows Print Server must be shared for it to display in Printer Installer. As such, the number listed in the Shared Printers column and the one in the Listed Printers column frequently match. When Printer Installer performs a synchronization with the Windows Print Server, the number of shared printers on the server will appear in this column.

Listed Printers

The number of printers (from the Windows print server) appearing in the tree view folder that was selected to display the printer objects corresponding to the printers on the server.

Errors

Errors do not occur often with Windows Print Server Links in Printer Installer. In the event that you do see an error notification, it is most commonly the Error Listing Printers, which occurs when the client on the Windows Print Server finds a change but cannot sync with Printer Installer. If no errors occur, then you will see a 0 in the Errors column.

List Printers In

This column shows the folder in the Admin Console tree view that displays the printers that the client finds on the Windows Print Server. This column is populated when you click Add and then click Browse in the List Printers In field as shown below.
