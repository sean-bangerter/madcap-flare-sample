Map Tab Cheat Sheet

The Map tab displays an image of the location (a blueprint) in which the printers

The Map tab enables the administrator to apply a blueprint image of a geographical location to a folder in the Admin Console tree view and then drag-and-drop printer objects from the same folder onto the map.

When the end-user logs in to the Self-service Portal and selects the same folder, then the map will appear. The end-user can click a printer object on the map to install the drivers for that printer object. Let's put this into context. Read More...

The folder you select in the tree view is the folder to which the map will be associated. If you select another folder, a different map (or no map at all) will appear in the Map tab. For this reason, the tree view is recommended to be set up by location. For example, if your office is on the fourth floor of your building, then the tree view folder would be named Fourth Floor and a map that shows the layout of the fourth floor is added to the Map tab (making sure you have the Fourth Floor folder selected in the tree view. Furthermore, the printer objects in the Fourth Floor Folder will be available as printer icons in the Map tab as well, the purpose of which is to drag-and-drop those icons into the location on the map that represents their actual location on the Fourth Floor.

Both the folder (assuming it has not been set to be hidden in the General tab) and the map will appear in the Self-service Portal. The end-user can click the icon and easily install the printer. The fields of the Map Tab are described in detail below.

Upload Background Image

Click this option to find the image you want to upload to the Map tab. The image you add will apply only to the selected folder in the tree view, and will appear when you select the same folder in the Self-service Portal. For example, if you select the New York folder in the Admin Console tree view and then upload a map in the Map tab, then when you select the New York folder in the Self-service Portal, you will see the map you uploaded as shown below.

In the Admin Console Map tab, only one map can be uploaded to each folder in the tree view. The Upload Background Image link is located in two spots of the Map tab - in the header and in the middle of the tab. Either option will enable you to upload a map. When a map has been uploaded, the option in the header will be replaced with the Remove Background Image link.

Remove Background Image

This option appears only if you have uploaded an image to the Map tab. Click Remove Background Image if you would like to delete the image that is currently appearing in the Map tab.

Printer List

The Printer List is the gray box that appears to the left of the map. When you select a folder in the Admin Console tree view, you will notice that the printer object(s) contained in the folder also appear in the Printer List box. The purpose of this box is to enable you to drag-and-drop printer objects onto the map. Any printer objects you apply to a map will appear on the Map in the Self-service Portal.

For example, in the Admin Console tree view, you select New York and then click the Map tab, which has Printer A appearing in the Printer List box. You drag-and-drop the Printer A icon onto the map and save your changes. When you open the Self-service Portal and click the New York folder in the tree view, the map will appear with Printer A in its designated location. You can click the icon to install the drivers for that printer.
