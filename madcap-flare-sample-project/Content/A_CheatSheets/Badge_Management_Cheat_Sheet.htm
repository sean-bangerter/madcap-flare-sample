Badge Management Cheat Sheet

The Badge Management settings appear in the Admin Console when you click Tools > Badge Management. The image to the left shows the Badge Management settings as they appear in Printer Installer.

The Badge Management settings not only display the badge user information and badge ID, but they also provide the administrator with other options including Register New, used for configuring Simple Badge Release Devices, Unregister for removing registered badges from the database, and Export for exporting badge usernames and IDs to a .csv file. The Badge Management setting also provide a Refresh button to manually update the list to match the badges in the database. The fields of the Badge Management settings are described in detail below.

Register New

The Register New button opens the Register Badges window, as shown below.

NOTE: Before you proceed, make sure you have one of the supported Simple Badge Release devices connected to the network and a card reader connected to the Simple Badge Release device. Also, make sure you have an LDAP domain set up in Tools > General under the LDAP Authentication Settings.

The Register Badges window will enable you to first configure the Simple Badge Release device after which you can swipe badges and register them directly in Printer Installer.

IP Address

In this field, enter the IP address of the Simple Badge Release device you are using to register badges. Printer Installer will use the IP address in this field to connect to and configure the Simple Badge Release device.

Configure RFIDeas Ethernet 241

NOTE: Before you can use this button, you must have a supported Simple Badge Release device connected to the network and a badge reader connected to the Simple Badge Release device. These items enable badge scanning in Printer Installer.

This button enables you to connect to and configure a supported RFIDeas Simple Badge Release device. Before you click this button, you must provide the device's IP address in the IP Address field.

When Printer Installer is finished configuring the Simple Badge Release device, the device will restart automatically. You will receive a message if the configuration is successful.

Configure Elatec TCPConv 2

NOTE: Before you can use this button to register badges, you must have a supported Simple Badge Release device connected to the network and a badge reader connected to the Simple Badge Release device. These items enable badge scanning in Printer Installer.

This button enables you to connect to and configure a supported Elatec Simple Badge Release device. Before you click this button, you must provide the device's IP address in the IP Address field.

When Printer Installer is finished configuring the Simple Badge Release device, the device will restart automatically. You will receive a message if the configuration is successful.

Badge ID

When you swipe a badge over the badge reader to register the badge, the badge's ID will appear in this field. The badge IDs are hard-coded into the badges themselves, so this field is read-only. The badge ID will appear automatically in the Badge ID column as shown below.

Badge AD User Name

Use this field to enter the domain and Active Directory credentials of the end-user who will be using the badge to release print jobs.

NOTE: The domain must be entered in all caps followed by a backslash and the Active Directory username. For example, PRINTERLOGIC\John.Smith.

Register Badge

After you have swiped a badge over the badge reader to register it, and you enter the domain and Active Directory username, click Register Badge to begin the badge registration process.

Delete Badge

This feature is useful if you have swiped a badge, and it appears in the Register Badges screen, but you do not want to register it (for whatever reason). You can delete a swiped badge without having to go through the registration process before you delete it.

Unregister

When you select a registered badge from the list, the Unregister button becomes available, enabling you to unregister a registered badge. For Printer Installer, the badge can be used for the Control Panel Application and for Simple Badge Release. When a badge is unregistered, it can no longer be used to release print jobs.

Export

Click this option to export the list of registered badges in Printer Installer to a .csv file. The process to save the csv file will be triggered to run automatically. Depending on the browser you are using, the interface will vary. This feature is generally used for exporting badges when you want to use the badges in a system other than Printer Installer.

Refresh

The Refresh button in the Badge Management section because newly registered badges do not always appear in the list of badges. Just click Refresh, and all badges (including the new ones) should appear in the list.

Badge User Name

When you registered a new badge (by clicking Register New), you were required to provide the domain followed by a backslash (\) and your Active Directory username. After the badge is registered, that domain and username information is added to this column.

Badge ID

When you register a new badge, you are required to swipe the badge over the card reader to add it to the system. The card contains a hard-coded, unique ID that Printer Installer picks up when the card is swiped over the card reader. The ID is shown, read-only, in this column to enable the administrator to quickly identify the person to whom the card is registered.

Last Login

This section shows the last time the end-user swiped the badge over the badge reader.

Settings Icon

The Settings Icon enables you to configure the way in which the Badge Management section displays data. You can set the width of the table that displays the data, the number of badges that displays on each page, and the frequency with which Printer Installer refreshes the table.
